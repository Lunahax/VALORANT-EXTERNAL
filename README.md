## <a id="features"></a>🌸 〢 Features

```
AIMBOT
- 7 different aimbot modes
  - Unsilent / Silent / Smooth / Assist / Correction / Triggerbot / Colorbot
- 3 different toggle modes
  - On ADS / On key press / On key hold
- Custom accuracy
- Custom aimbot FOV

ESP
- 3 ESP modes
  - Custom box colors
- Toggle Tracers
  - Custom tracer colors

VISUALS
- Wireframe
- Skin changer
- Unlock skins
- Unlock Buddies
- Unlock Player cards
```
